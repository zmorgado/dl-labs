# Deep Learning Module

This repository contains Deep Learning theory notebooks and hands-on exercises, organized to match the ML module workflow.

## Structure

- `theory/`: conceptual notebooks
- `exercises/`: practical labs and exercises
- `datasets/`: data files used by notebooks (`.csv`, `.h5`)
- `images/`: static images referenced by notebooks

### Theory
- `theory/deep_learning_1.ipynb`
- `theory/dcgan.ipynb`

## Setup

1. Create and activate a virtual environment:
   - macOS/Linux:
     - `python3 -m venv .venv`
     - `source .venv/bin/activate`

2. Install dependencies:
   - `pip install -r requirements.txt`

3. Launch Jupyter:
   - `jupyter notebook`

4. Run notebooks from their folder locations (`theory/` or `exercises/`).
